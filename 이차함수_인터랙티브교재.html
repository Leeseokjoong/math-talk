개념 1
이차함수의 뜻
땡중샘

태린아, 함수 기억나? y가 x에 의해 딱 하나로 결정되는 관계.

태린

네! 일차함수에서 배웠어요. y = ax + b 형태요.

땡중샘

맞아. 이번엔 x가 2승으로 올라가. y = ax² + bx + c 형태, 이게 이차함수야.

y = ax² + bx + c   (a ≠ 0)

a ≠ 0이어야 이차함수야. a = 0이면 그냥 일차함수가 돼버려.

태린

y = x²도 이차함수예요?

땡중샘

응! a=1, b=0, c=0인 가장 기본적인 이차함수야. x에 숫자를 넣어볼게.

x−3−2−10123
y=x² 941 0 149

x = −2와 x = 2일 때 y 값이 같아. 대칭이야!

땡중샘

확인 문제!

CHECK
다음 중 이차함수인 것은?
땡중샘

①은 일차함수, ②는 x²이 소거돼서 y=2x 일차, ④는 분수함수. ③만 이차함수야!

이차함수
y = ax² + bx + c  (a ≠ 0)
x²의 계수 a가 0이 되지 않는지 꼭 확인!
수정 요청 메모
개념 2
y = ax²의 그래프
땡중샘

y = x² 값표의 점들을 이으면 곡선이 나와. 이 곡선을 포물선이라고 해.

꼭짓점은 (0, 0), 축은 x = 0 (y축). 아래로 볼록한 모양이야.

태린

a가 바뀌면 모양이 어떻게 달라져요?

땡중샘

a의 절댓값이 클수록 폭이 좁아지고, 작을수록 넓어져. 비교해볼게.

y = 2x²
y = x²
y = ½x²
땡중샘

a가 음수이면 위로 볼록해. y = −x²를 보면 y = x²을 x축 기준으로 뒤집은 모양이야.

y = x² (아래 볼록)
y = −x² (위로 볼록)
땡중샘
y = ax² 그래프 정리
꼭짓점: (0, 0)  ·  축: x = 0
a > 0 → 아래로 볼록  |  a < 0 → 위로 볼록
|a| 클수록 폭 좁음  |  |a| 작을수록 폭 넓음
땡중샘

확인 문제!

CHECK
y = −3x²의 그래프에 대한 설명으로 옳은 것은?
땡중샘

a=−3이니까 위로 볼록, |a|=3으로 y=x²보다 폭이 좁아. y=3x²와는 x축 대칭으로 모양은 같지만 방향이 반대야!

수정 요청 메모
개념 3
y = a(x−p)² + q 평행이동
땡중샘

y = ax²를 좌우로 p, 위아래로 q만큼 움직인 게 y = a(x−p)² + q야.

y = a(x−p)² + q
꼭짓점: (p, q)  ·  축: x = p
y = ax²을 x축 방향 p, y축 방향 q만큼 평행이동

태린

y = (x−2)² + 1이면 꼭짓점이 (2, 1)이에요?

땡중샘

딱 맞아! 그래프로 보면 이래.

y = x²
y = (x−2)²+1
땡중샘

주의! 부호 헷갈리지 말자.

y = (x − 2)² + 1 → 꼭짓점 (+2, 1)  오른쪽으로 이동
y = (x + 3)² − 1 → 꼭짓점 (−3, −1)  왼쪽으로 이동

괄호 안 부호와 꼭짓점 x좌표 부호가 반대야!

땡중샘

확인 문제!

CHECK
y = 2(x+1)² − 3의 꼭짓점 좌표는?
땡중샘

y = a(x−p)²+q에서 p = −1, q = −3. x+1 = x−(−1)이니까 꼭짓점은 (−1, −3)!

수정 요청 메모
개념 4
y = ax² + bx + c → 표준형 변환
땡중샘

y = ax² + bx + c 형태로 주어지면 꼭짓점을 바로 못 읽어. 완전제곱식으로 바꿔야 해.

ax² + bx + c → a(x−p)² + q 로 변환
이 과정을 표준형으로 변환한다고 해.

땡중샘

y = x² − 4x + 7을 표준형으로 바꿔볼게.

출발
y = x² − 4x + 7
x항 묶기
y = (x² − 4x) + 7
+4 보완
y = (x² − 4x + 4 − 4) + 7
완전제곱
y = (x − 2)² − 4 + 7
표준형
y = (x − 2)² + 3

꼭짓점: (2, 3), 축: x = 2

태린

+4를 더했다가 왜 −4를 또 빼요?

땡중샘

+4를 더하면 식의 값이 바뀌잖아. 그래서 −4도 같이 써서 값이 안 바뀌게 하는 거야. 더하고 빼기 — 이게 핵심 트릭이야!

x의 계수가 b일 때, (b/2)²을 더하고 빼면 완전제곱식이 만들어져.
x² − 4x + □ = (x−2)²  →  □ = (−4÷2)² = 4

땡중샘

a ≠ 1일 때는 먼저 a로 묶어야 해. y = 2x² − 8x + 6 예제.

출발
y = 2x² − 8x + 6
2로 묶기
y = 2(x² − 4x) + 6
+4−4 보완
y = 2(x² − 4x +4−4) + 6
2 분배
y = 2(x−2)² − 8 + 6
표준형
y = 2(x−2)² − 2  →  꼭짓점 (2, −2)
땡중샘

확인 문제!

CHECK
y = x² + 6x + 5를 표준형으로 변환하면?
땡중샘

x² + 6x + 9 − 9 + 5 = (x+3)² − 4. (6÷2)² = 9를 더하고 빼면 돼!

개념 5
이차함수의 식 구하기
땡중샘

그래프를 보고 식을 거꾸로 구해야 하는 문제야. 두 가지 방법이 있어.

방법 1
꼭짓점과 다른 점 1개를 알 때 → y = a(x−p)² + q 이용
방법 2
세 점을 알 때 → y = ax² + bx + c에 대입
땡중샘

방법 1 예제
꼭짓점이 (1, 2)이고 점 (3, 10)을 지나는 이차함수의 식을 구해.

꼭짓점 대입
y = a(x−1)² + 2
(3, 10) 대입
10 = a(3−1)² + 2 = 4a + 2
a 계산
4a = 8 → a = 2
정답
y = 2(x−1)² + 2
땡중샘

방법 2 예제
(0, 1), (1, 0), (2, 3)을 지나는 이차함수의 식을 구해.

y=ax²+bx+c
세 점을 각각 대입해서 연립
(0,1) 대입
c = 1
(1,0) 대입
a + b + 1 = 0 → a + b = −1
(2,3) 대입
4a + 2b + 1 = 3 → 4a + 2b = 2
연립 풀기
a = 2, b = −3
정답
y = 2x² − 3x + 1
땡중샘

확인 문제!

CHECK
꼭짓점이 (−2, 3)이고 점 (0, 7)을 지나는 이차함수의 식은?
땡중샘

y = a(x+2)² + 3에 (0, 7) 대입: 7 = a(4) + 3 → 4a = 4 → a = 1.
y = (x+2)² + 3!
