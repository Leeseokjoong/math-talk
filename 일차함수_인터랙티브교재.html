개념 1 — 함수의 뜻 0%
개념 1
함수란 무엇인가
땡중샘

태린아, 오늘부터 함수 시작이야. 갑자기 퀴즈 하나 — 함수가 뭔지 알아?

태린

음... 잘 모르겠어요. 함수라는 말은 들어봤는데 너무 어렵게 느껴져요.

땡중샘

당연히 낯설지. 근데 사실 엄청 간단해. 자판기를 생각해봐.

1번 버튼 누르면 → 콜라가 나와. 딱 하나야.

1번 눌렀는데 콜라도 나오고 사이다도 나오고 물도 나오고... 이러면 고장난 자판기잖아.

수학에서도 마찬가지야. x 하나 넣으면 y 하나만 나오는 것 — 그게 함수야.

태린

아, 그러면 x 하나에 y 하나만 나오면 함수인 거예요?

땡중샘

정확해! 앞으로 함수가 뭐냐고 물으면 딱 이렇게 대답해.

"x 하나당 y 하나가 결정되는 관계"

이것만으로도 함수의 개념을 설명할 수 있는 사람이 되는 거야.

태린

선생님, 그럼 자연수 4의 약수 y는요? 이건 함수예요?

땡중샘

좋은 예시야! 4의 약수는 1, 2, 4 — 이렇게 여러 개의 답이 나오니까 그것은 함수관계라고 할 수 없어.

하지만 4의 약수의 개수는? — 3개라고 답을 하나로 말할 수 있으니 함수관계가 맞아.

함수 ✓
y = 2x + 1
x = 3 → y = 7만 나와
y가 딱 하나
함수 ✗
자연수 x의 약수 y
x = 4 → y = 1, 2, 4
y가 여러 개 → 함수 아님
함수 ✓
자연수 x의 약수의 개수 y
x = 4 → y = 3 (딱 하나)
y가 하나만 나와
함수 ✗
자연수 x보다 작은 소수 y
x = 10 → y = 2, 3, 5, 7
y가 여러 개 → 함수 아님
태린

"약수의 개수"는 하나니까 함수고, "약수" 자체는 여러 개니까 함수가 아닌 거군요. 말 하나 차이인데 완전 다르네요.

땡중샘

정확히 짚었어. 그럼 실력 확인 한번 해보자.

확인 문제
다음 중 y가 x의 함수인 것은?
땡중샘

이번엔 f(x) 표기법이야. 함수는 영어로 function이라고 해. 그래서 f라고 쓰는 거야.

태린

f(x)요? 선생님, 이거 너무 어려워요. 갑자기 f가 왜 나와요? 처음 보는 기호예요.

땡중샘

처음 보면 낯설어 보이지. 근데 하나씩 뜯어보면 쉬워.

f → function (함수)의 줄임말
x → 우리가 집어넣는 값
f(x) → x를 넣었을 때 나오는 값 = 함수값

자판기에 x를 넣으면 f(x)가 나와. f는 자판기 자체(규칙)야.

태린

그러면 그냥 y라고 하면 되잖아요? y = x + 1 이나 f(x) = x + 1 이나 같은 거 아닌가요?

땡중샘

계산 결과는 같아. 근데 바라보는 관점이 달라.

y = x + 1 의 관점

y는 그냥 결과값의 이름이야. "x와 y 사이에 이런 관계가 있어" — 거기서 끝.

f(x) = x + 1 의 관점

f라는 함수는 x를 집어넣으면 x+1을 내보내는 규칙이야. 값이 아니라 기계 자체를 표현하는 거야.

땡중샘

f(x) 표기를 쓰면 두 가지를 동시에 말할 수 있어. "이건 함수야" + "x를 넣으면 이 값이 나와".

그래서 f(3)이라고 쓰면 "x = 3일 때 함수값"이 돼. 짧고 명확하지.

태린

아, f(3)이라고 쓰면 x에 3을 넣으라는 거예요? 그러면 계산만 하면 되겠네요!

땡중샘

맞아! 그냥 x 자리에 숫자 집어넣으면 끝이야.

f(x) = 2x + 1 일 때
f(3) = 2×3 + 1 = 7
f(−1) = 2×(−1) + 1 = −1
땡중샘

연습 문제 몇 개 풀어봐.

확인 문제
x + 1 = y 에서 y가 의미하는 것과 다른 것은?
확인 문제
f(x) = 2x + 1 일 때, f(1)의 값은?
확인 문제
f(x) = 2x + 1 일 때, f(−3)의 값은?
땡중샘

마지막으로 하나 더. x + 1 이라는 똑같은 식도 어떻게 쓰느냐에 따라 이름이 달라져.

x + 1
일차식그냥 식. 등호도 부등호도 없어.
x + 1 = 0
방정식등호(=)로 연결. x의 특정 값을 구해.
x + 1 > 0
부등식부등호로 연결. x의 범위를 구해.
y = x + 1
함수식x와 y의 관계. x마다 y가 하나씩 결정돼.
태린

같은 x + 1 인데 y = 이 붙으면 함수식, = 0이 붙으면 방정식이 되는 거군요. 이름 외우는 것보다 이게 훨씬 쉬운데요?

땡중샘

바로 그거야! 무엇과 연결하느냐가 이름을 결정해. 다음 넘어가기 전에 참고 박스 봐. 1학년 때 배운 순서쌍이랑 좌표평면이 기억 안 나면 여기서 복습해.

순서쌍 (x, y)

x좌표와 y좌표를 순서대로 묶은 쌍. (3, 5)라고 쓰면 x = 3, y = 5야. 순서가 바뀌면 다른 점이 돼 — (5, 3) ≠ (3, 5).

좌표평면
사분면

좌표평면은 x축과 y축으로 4개 영역으로 나뉘어. 제1사분면(+,+), 제2사분면(−,+), 제3사분면(−,−), 제4사분면(+,−).

수정 요청
개념 2
일차함수의 뜻
땡중샘

태린아, 일차함수가 뭔지 말해봐.

태린

음... 함수인데 일차인 거요? y = ax + b 같은 거 아닌가요?

땡중샘

거의 맞았어! 정확한 정의는 이래.

일차함수 정의

y = ax + b (a ≠ 0, a, b는 상수) 형태로 나타내어지는 함수를 y는 x의 일차함수라 한다.

태린

그런데 왜 a ≠ 0이어야 해요? a가 0이면 어떻게 되는 거예요?

땡중샘

좋은 질문이야. a = 0이면 y = b가 돼. 그러면 x가 1이든 100이든 −5든 상관없이 y는 항상 b야.

x가 y를 결정하는 게 아니라 — 그냥 숫자 하나만 있는 거야. 그래서 상수함수라고 해. 일차함수가 아니야.

태린

그럼 y = 4는 일차함수가 아닌 거네요. 판별하는 방법이 있어요?

땡중샘

세 가지만 확인하면 돼.

① x의 차수가 1이어야 해 (x², x³ 안 됨)
② 분모에 x가 없어야 해 (1x 안 됨)
③ a ≠ 0 이어야 해 (y = 4 같은 상수함수 안 됨)

일차함수 ✓
y = 3x − 5
a=3, b=−5, a≠0
일차함수 ✓
y = −x + 2
a=−1, b=2, a≠0
일차함수 ✗
y = x² + 1
x차수가 2 → 이차함수
일차함수 ✗
y = 1x
분모에 x 있음
일차함수 ✗
y = 4
a = 0 → 상수함수
일차함수 ✓
y = x
a=1, b=0, a≠0
태린

선생님, y = 1x 은 왜 안 되는 거예요? 너무 어렵게 생겼어요.

땡중샘

x = 0이라고 해봐. 분모가 0이 되잖아. 수학에서 0으로 나누는 건 정의가 안 돼. 그리고 y = ax + b 형태로 쓸 수도 없어.

이건 나중에 배울 반비례 관계야. 지금은 "분모에 x 있으면 일차함수 아님" 이것만 기억해.

땡중샘

f(x) = 2x − 3 일 때 값 계산은 전에 배운 것처럼 x 자리에 수를 집어넣으면 돼. 확인해봐.

f(2)2 × 2 − 3 = 1
f(−1)2 × (−1) − 3 = −5
f(0)2 × 0 − 3 = −3  ← y절편이야
땡중샘

확인 문제!

확인 문제
f(x) = −2x + 5 일 때 f(3)의 값은?
땡중샘

일차함수의 뜻 완료! 이제 그래프로 넘어가자.

일차함수의 그래프
땡중샘

y = 2x + 1 을 좌표평면에 그리려면 어떻게 할 것 같아?

태린

x에 숫자를 넣어서 y를 구하고 점을 찍으면 되지 않아요?

땡중샘

맞아! 한번 해보자. x에 −2부터 2까지 넣어봐.

x−2−1012
y = 2x+1 −3 −1 1 3 5
땡중샘

이 점들을 좌표평면에 찍으면 어떻게 될까?

태린

점들이 일직선으로 늘어서네요! 그럼 점을 엄청 많이 찍어야 하는 거예요?

땡중샘

아니야! 점 2개면 충분해. 직선은 두 점만 알면 그을 수 있거든.

y절편 (0, 1)에 점 찍고 → (1, 3)에 점 찍고 → 두 점 연결. 끝이야.

땡중샘

순서대로 정리하면 이래.

1
y절편을 구한다
x = 0 대입 → y = b. 점 (0, b) 찍기
2
점 하나를 더 구한다
x = 1 대입 → y = a + b. 점 (1, a+b) 찍기
3
두 점을 지나는 직선을 긋는다
양쪽으로 쭉 연장해야 해. 선분이 아닌 직선이야.
태린

그냥 y절편이랑 점 하나 더 찍으면 되는 거네요. 생각보다 간단하다!

순서쌍 (x, y)

(2, 3)과 (3, 2)는 달라 — 순서가 중요해서 "순서쌍"이야.

좌표평면

x축(가로)과 y축(세로)이 원점 O(0,0)에서 만나는 평면이야. 점 (a, b)는 x방향으로 a, y방향으로 b만큼 이동한 위치.

수정 요청
개념 4
기울기와 y절편 · x절편
땡중샘

y = ax + b 에서 a랑 b가 각각 뭘 담당하는지 알아?

태린

a가 기울기고 b가 y절편이요?

땡중샘

맞아! 그런데 왜 b가 y절편인지 먼저 알아야 해. 그래프 한번 봐봐. 직선이 y축이랑 만나는 점 보여?

태린

y축에 찍힌 초록 점이요? (0, 4) 라고 써있네요.

땡중샘

잘 봤어! 그런데 y축 위에 있는 점들은 x좌표가 항상 얼마야? 원점 (0, 0), 저 점 (0, 4), y축 위 다른 점들... 공통점이 뭐야?

태린

다 x = 0 이잖아요! y축에 있으면 오른쪽이나 왼쪽으로 안 움직이니까요.

땡중샘

바로 그거야! y절편은 y축 위의 점이니까 x = 0이고, 그래서 y절편을 구하려면 x = 0을 대입하는 거야. 확인해 볼까?

식y = 2x + 4
x = 0 대입y = 2 × 0 + 4 = 4
y절편점 (0, 4) — 그래프의 초록 점과 일치!
태린

아! y축에 있으니까 x = 0인 거고, 그래서 x = 0을 대입하는 거군요. 이유가 있었네요.

땡중샘

그럼 x절편은? 다시 그래프 봐. 직선이 x축이랑 만나는 점 보여?

태린

x축에 찍힌 주황 점이요? (−2, 0) 이네요.

땡중샘

x축 위의 점들은 y좌표가 항상 얼마야? (0, 0), (−2, 0), x축 위 다른 점들...

태린

다 y = 0이요! x축에 있으면 위아래로 안 움직이니까요.

땡중샘

완벽해! x절편도 같은 원리야. x축 위의 점이니까 y = 0이고, 그래서 y = 0을 대입해서 구해.

식y = 2x + 4
y = 0 대입0 = 2x + 4  →  2x = −4  →  x = −2
x절편점 (−2, 0) — 그래프의 주황 점과 일치!

y절편: y축 위의 점 → x = 0 대입 → 점 (0, b)

x절편: x축 위의 점 → y = 0 대입 → 점 (−ba, 0)

태린

y절편은 y축 위 → x=0, x절편은 x축 위 → y=0. 이름이랑 반대처럼 보이지만 이유가 있었군요!

땡중샘

이번엔 기울기야. 기울기의 부호는 방향으로 결정돼. 왼쪽에서 오른쪽으로 읽을 때, 올라가면 +, 내려가면 −이야.

a > 0
우상향
(오른쪽 위 방향)
a < 0
우하향
(오른쪽 아래 방향)
태린

땡중샘, 그런데 좌측으로 올라가는 것은 왜 +기울기가 아니에요? 좌상향은 안 되는 거예요?

땡중샘

응, 우리 태린이가 왼손잡이인 것을 샘이 깜빡했구나.

안타깝게도 수학은 오른손잡이들 위주로 발전하다 보니, 그래프도 왼쪽 → 오른쪽으로 흘러가는 방향을 기준으로 삼게 됐어. 좌상향도 그래프 방향을 반대로 읽으면 '우하향'이 되거든.

그래서 어느 방향으로 보든, 왼쪽에서 오른쪽으로 읽을 때 올라가면 +, 내려가면 −로 약속한 거야.

태린

방향은 알겠는데 기울어진 정도를 어떻게 숫자로 나타내요? 각도로 하면 안 되나요?

땡중샘

각도로도 나타낼 수 있어! 30°, 45°, 60° 기울기를 한 그래프에 그려봤어. 각도가 클수록 더 가파르지?

태린

확실히 60°가 제일 가파르고 30°가 제일 완만하네요. 그러면 각도로 표현하면 되지 않나요?

땡중샘

아쉽게도 각도는 문제가 있어. 그래프 종이를 크게 프린트하면 같은 직선도 각도가 달라 보이거든. 축의 단위가 바뀌면 각도도 바뀌어. 그래서 수학에서는 좌표의 비율로 표현해 — x가 1 늘 때 y가 얼마나 변하는지야. y = x 랑 y = 2x 를 비교해봐.

y = x (기울기 1/1)
y = 2x (기울기 2/1)
태린

y = x 는 x가 1 늘면 y도 1 늘고, y = 2x 는 x가 1 늘면 y가 2 늘어요. 비율로 나타내는 거군요!

땡중샘

맞아! 그걸 식으로 정리하면 이래. y끼리 뺀 것x끼리 뺀 것 — 이렇게 외우면 절대 안 잊어.

기울기 a = y의 변화량x의 변화량 = y₂ − y₁x₂ − x₁

y = 2x 위의 두 점 (1, 2) → (3, 6)
a = 6 − 23 − 1 = 42 = 2
땡중샘

그런데 왜 y끼리 빼고 x끼리 빼는 걸까? 직각삼각형으로 보면 딱 이해돼. 그래프 봐봐.

태린

아! 두 점 사이에 직각삼각형이 생기는군요. 가로가 x의 차이고 세로가 y의 차이예요!

땡중샘

맞아! 기울기 = 높이 (y의 차이)밑변 (x의 차이) 야.
그래서 분자에 y끼리 뺀 것, 분모에 x끼리 뺀 것이 오는 거야.

직각삼각형에서

· 밑변 = x₂ − x₁ (오른쪽으로 얼마나 이동했나)

· 높이 = y₂ − y₁ (위로 얼마나 이동했나)

기울기 = 높이밑변 = y₂ − y₁x₂ − x₁

땡중샘

그리고 부호 실수를 막는 꿀팁이 있어! 뺄 때 항상 같은 방향으로 빼야 해.

두 점(1, 2) 와 (3, 6)
올바름6 − 23 − 1 = 42 = 2   (둘 다 ②번 − ①번)
올바름2 − 61 − 3 = −4−2 = 2   (둘 다 ①번 − ②번)
틀림6 − 21 − 3 = 4−2 = −2   (방향이 섞임!)

규칙: y 분자를 (두 번째 − 첫 번째)로 빼면, x 분모도 (두 번째 − 첫 번째)로. 둘 다 같은 방향!

태린

그런데 교과서마다 표현이 달라요. f(b) − f(a) / b − a 이런 식도 있던데 그건 뭐예요?

땡중샘

다 같은 말이야! 기울기를 표현하는 방법이 여러 가지가 있는 것뿐이야. 하나씩 비교해볼게.

기울기를 표현하는 다양한 방법
① 변화량 표현
y의 변화량x의 변화량  =  ΔyΔx
Δ(델타) = 변화량을 뜻하는 기호
② 두 점 좌표 표현
y₂ − y₁x₂ − x₁
두 점이 있을 때 각각 이름을 붙여:
첫 번째 점 → (x₁, y₁)  x₁ = 첫 번째 점의 x값, y₁ = 첫 번째 점의 y값
두 번째 점 → (x₂, y₂)  x₂ = 두 번째 점의 x값, y₂ = 두 번째 점의 y값
아래첨자 1, 2는 그냥 "첫 번째", "두 번째"라는 뜻이야.
예) 점 (1, 2) 와 (3, 6) 이면
x₁=1, y₁=2, x₂=3, y₂=6
→ y₂ − y₁x₂ − x₁ = 6 − 23 − 1 = 2
③ 함수 표기법
f(b) − f(a)b − a
f(x) = y이므로 y₂ − y₁ / x₂ − x₁ 와 똑같아

암기법   y끼리 뺀 것x끼리 뺀 것

분모·분자 위치만 기억하면 어떤 표현이 나와도 같은 말이야!

태린

다 y끼리 뺀 것x끼리 뺀 것 이네요! 표현만 다르고 의미는 하나구나.

땡중샘

맞아. 그리고 한 가지 더 — 뺄 때 순서를 바꿔도 돼. 분모·분자 둘 다 뒤집으면 결과가 같아.

두 점(1, 2) 와 (3, 6)
순서 ①6 − 23 − 1 = 42 = 2
순서 ② (뒤집기)2 − 61 − 3 = −4−2 = 2
주의

분모·분자를 둘 다 뒤집어야 해. 분자만 또는 분모만 뒤집으면 부호가 틀려!

땡중샘

a의 부호와 크기에 따라 y = ax 그래프 모양이 달라져.

y = 2x (a > 0, 우상향)
y = −2x (a < 0, 우하향)

|a|가 클수록 y축에 가깝고 가파라져.

|a|가 작을수록 x축에 가깝고 완만해져.

y = 3x (가파름)
y = ½x (완만)
태린

그런데 선생님, y = 2x 랑 y = 2x + 3이랑 y = 2x − 2 는 기울기가 다 똑같은데... 그래프가 어떻게 생겼어요?

땡중샘

좋은 질문이야! 직접 봐봐. a가 같으면 어떻게 되는지.

태린

세 직선이 서로 평행하네요! 모양은 똑같고 위아래로만 이동한 것 같아요.

땡중샘

정확해! 이걸 평행이동이라고 해. y = 2x 의 그래프를 y축 방향(위아래)으로 b만큼 이동하면 y = 2x + b 가 되는 거야.

y = ax 의 그래프를 y축 방향으로 b만큼 평행이동

→ y = ax + b

b > 0 이면 위로, b < 0 이면 아래로 이동

태린

그럼 a가 같은 두 일차함수는 항상 평행한 거네요? 절대 만나지 않고요.

땡중샘

맞아! 기울기가 같으면 평행이야. 딱 하나 예외는 b도 같으면 — 그건 완전히 똑같은 직선이 되어버려. 시험에 꼭 나오는 포인트야.

a가 같고 b가 다르면 → 두 직선은 평행 (만나지 않음)
a가 같고 b도 같으면 → 일치 (완전히 같은 직선)
태린

그러면 y = ax + b 에서 a랑 b 부호 조합에 따라 그래프 모양이 정해지겠네요?

땡중샘

a < 0   b > 0
