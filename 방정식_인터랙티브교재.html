문자와 식 — 대입의 뜻
땡중샘

태린아, 사과가 x개 있어. 한 개에 500원이면 전체 가격은 얼마야?

태린

500 곱하기 x니까 500x원이에요!

땡중샘

맞아. x가 3이면 1500원, x가 10이면 5000원이야. 이렇게 문자를 쓰면 수량을 일반적으로 나타낼 수 있어.

사과 x개, 가격 500x원  (x = 1, 2, 3, ...)
태린

그런데 왜 굳이 문자를 써요? 그냥 숫자로 쓰면 안 돼요?

땡중샘

숫자로 쓰면 딱 그 경우만 돼. x=3이면 1500원, x=10이면 5000원 — 두 개의 식을 따로 써야 해. 문자를 쓰면 모든 경우를 식 하나로 나타낼 수 있어. 이게 일반화의 힘이야.

대입(代入): 식에 있는 문자 대신 구체적인 수를 넣는 것.
그때 계산한 결과를 식의 값이라고 해.

땡중샘

예를 들어, x = 3일 때 2x + 1의 값은?

x = 3 대입2 × 3 + 1
계산6 + 1
식의 값7

x = 2일 때, 3x - 1의 값은?
수정 요청
개념 2
일차식과 동류항
땡중샘

식에서 중요한 용어 먼저 정리하자. 항은 수나 문자의 곱으로 이루어진 식이야. 3x에서 3은 계수, x가 문자야. 그리고 숫자만 있는 항을 상수항이라고 해.

항
3x, −2, 5y
수나 문자의 곱으로 이루어진 식
계수
3x에서 3
문자 앞에 곱해진 수
상수항
−2, 7
숫자로만 이루어진 항
차수
x²: 2차, x: 1차
문자가 곱해진 횟수
땡중샘

항이 하나인 식은 단항식, 둘 이상인 식은 다항식이야. 그리고 차수가 1인 다항식이 바로 일차식이야.

단항식: 3x
다항식: 3x + 2,  2x−5y+1
일차식: 2x+1,  x−3  (최고 차수 = 1)
땡중샘

이제 동류항! 문자와 차수가 같은 항을 동류항이라고 해. 동류항끼리는 계수를 합칠 수 있어.

동류항 합산3x + 2x = 5x  (x끼리)
이건 안 됨2x + 3y  (문자가 달라서 합칠 수 없음)
태린

그러면 2x²랑 3x도 동류항 아닌가요? 둘 다 x가 있잖아요.

땡중샘

예리한 질문! 동류항은 문자도 같고 차수도 같아야 해. 2x²는 2차, 3x는 1차 — 차수가 달라서 동류항이 아니야. 합칠 수 없어.

주의: 2x²와 3x는 동류항 X — 차수(2차 vs 1차)가 다르기 때문.

땡중샘

일차식의 덧셈/뺄셈 예시야. 동류항끼리 모아서 계산해.

원래 식(3x + 2) + (2x − 5)
동류항 모으기(3x + 2x) + (2 − 5)
결과5x − 3
땡중샘

개념 3
등식과 방정식
땡중샘

등호(=)가 들어간 식을 등식이라고 해. 등식에는 두 종류가 있어.

항등식
x + x = 2x
x에 어떤 수를 넣어도 항상 참인 등식
방정식
x + 1 = 4
특정 x 값을 넣을 때만 참이 되는 등식
땡중샘

방정식에서 값을 아직 모르는 문자를 미지수라고 해. 방정식을 참이 되게 하는 미지수의 값을 해(근)라고 해.

해(근): 방정식에 대입하면 등식이 성립하는 x의 값
x + 1 = 4의 해: x = 3  (3 + 1 = 4 성립)

태린

x + 1 = 4에서 x = 3이 해인지 어떻게 알아요?

땡중샘

직접 대입해서 확인하면 돼. x = 3을 왼쪽(좌변)에 넣어봐.

x = 3 대입좌변: 3 + 1 = 4
우변4
판정좌변 = 우변  → 해가 맞다!
땡중샘

개념 4
등식의 성질과 이항
땡중샘

방정식을 풀려면 먼저 등식의 성질을 알아야 해. 천칭 저울을 생각해봐. 양쪽이 균형을 이루고 있을 때, 양쪽에 같은 무게를 더하거나 빼도 균형은 유지되지?

땡중샘

등식도 마찬가지야. 4가지 성질이 있어.

1
양변에 같은 수를 더해도 등식이 성립한다.
a = b  →  a + c = b + c
2
양변에서 같은 수를 빼도 등식이 성립한다.
a = b  →  a − c = b − c
3
양변에 같은 수를 곱해도 등식이 성립한다.
a = b  →  ac = bc
4
양변을 같은 수(0 제외)로 나눠도 등식이 성립한다.
a = b  →  a ÷ c = b ÷ c (c ≠ 0)
땡중샘

이 성질을 이용하면 이항을 할 수 있어. 한 항을 부호를 바꿔서 반대편으로 옮기는 거야.

원래 식3x + 2 = 8
양변 − 23x = 8 − 2
계산3x = 6  (+2가 −2로 이항됨)
x 구하기x = 2
태린

이항할 때 왜 부호가 바뀌어요?

땡중샘

등식의 성질 때문이야. 3x + 2 = 8에서 양변에서 2를 빼면, 왼쪽에 있던 +2가 없어지고 오른쪽에는 −2가 붙어. 즉, 이항 = 양변에서 그 수를 빼거나 더하는 것이야.

이항의 법칙: 한 항이 등호를 건너가면 부호가 반대로 바뀐다.
+ 는 − 로, − 는 + 로, × 는 ÷ 로, ÷ 는 × 로.

땡중샘

일차방정식 풀기
땡중샘

일차방정식 풀이에는 정해진 순서가 있어. 이 순서를 따르면 어떤 문제도 풀 수 있어.

1
괄호 풀기 — 분배법칙 사용
2
이항 — x항은 좌변, 상수항은 우변으로
3
동류항 정리 — 같은 문자끼리 합치기
4
양변을 x의 계수로 나누기 — x = ? 완성
땡중샘

예제를 같이 풀어보자. 2(x − 1) = x + 3

① 괄호 풀기2x − 2 = x + 3
② 이항2x − x = 3 + 2
③ 동류항x = 5
해x = 5
땡중샘

계수가 분수일 때는 양변에 분모의 최소공배수를 곱해서 정수로 만들어. 예를 들어 12x + 1 = 34x − 12이면, 분모의 LCM인 4를 곱해.

양변 × 42x + 4 = 3x − 2
이항2x − 3x = −2 − 4
해x = 6
땡중샘

풀고 나서 검산은 필수야. 구한 x 값을 원래 식에 대입해서 등식이 성립하면 정답!

검산 방법: 구한 x 값을 원래 방정식의 좌변과 우변에 각각 대입해서 좌변 = 우변이 되면 해가 맞다.

땡중샘

땡중샘

문장으로 주어진 문제를 방정식으로 풀려면 4단계를 따라가면 돼.

1
미지수 정하기 — 구하려는 것을 x로 놓는다
2
방정식 세우기 — 문제의 조건을 식으로 나타낸다
3
풀기 — 일차방정식 풀이 절차대로
4
검산 — 해가 문제의 조건에 맞는지 확인
땡중샘

예제 1: 연속하는 두 정수의 합이 25일 때, 두 정수를 구하여라.

① 미지수작은 수를 x로 놓으면, 연속하는 두 정수는 x, x+1
② 방정식x + (x+1) = 25
③ 풀기2x + 1 = 25  →  2x = 24  →  x = 12
④ 답두 정수는 12와 13  (합 = 25 확인)
태린

미지수를 뭘로 놓아야 하는지 헷갈려요. 어떻게 정해요?

땡중샘

구하는 것을 x로 놓으면 돼. "두 정수를 구하여라" → 그 중 하나를 x로. 나머지는 x를 이용해서 나타내면 돼. 모르는 것을 x로 두고 아는 조건으로 식을 만드는 거야.

땡중샘

예제 2: 현재 아버지 나이는 아들 나이의 3배이다. 10년 후에는 아버지 나이가 아들 나이의 2배가 된다. 현재 아들의 나이는?

① 미지수현재 아들 나이 = x, 아버지 나이 = 3x
② 방정식10년 후: (3x+10) = 2(x+10)
③ 풀기3x+10 = 2x+20  →  x = 10
④ 답현재 아들 나이 10세, 아버지 30세  (10년 후: 40세 = 20세 × 2)
