개념 1
이차방정식의 뜻과 표준형
땡중샘

태린아, 어떤 수를 제곱하면 9가 된대. 무슨 수일까?

태린

3이요! 3² = 9니까요.

땡중샘

3 맞아. 근데 −3도 맞아. (−3)² = 9거든. 그래서 해가 두 개야.

이걸 방정식으로 쓰면 x² = 9, 이항하면 x² − 9 = 0. 이게 바로 이차방정식이야.

태린

일차방정식이랑 뭐가 달라요?

땡중샘

일차방정식은 x가 최고 1승, 이차방정식은 x²이 반드시 있어야 해.

ax² + bx + c = 0   (a ≠ 0)

이게 이차방정식의 표준형이야. 우변을 0으로 만든 형태.

태린

그럼 x² + 3x = 2도 이차방정식이에요?

땡중샘

응, 맞아! 표준형으로 바꾸려면 우변을 왼쪽으로 이항하면 돼.

원래 식
x² + 3x = 2
표준형
x² + 3x − 2 = 0
땡중샘

주의할 게 있어. 이차방정식처럼 보여도 아닌 경우가 있어.

x(x+1) = x² + 3을 정리하면?
x² + x = x² + 3 → x = 3 — x²이 소거되니까 일차방정식이야!

표준형으로 정리했을 때 x²의 계수 a ≠ 0이어야 이차방정식.

땡중샘

확인 문제 하나!

CHECK
다음 중 이차방정식인 것은?
땡중샘

①은 일차방정식, ②는 x²이 소거돼서 2x+1=0 일차, ④는 삼차방정식. ③만 이차야!

이차방정식이란
ax² + bx + c = 0 형태 (a ≠ 0)
정리 후에도 x²이 남아 있는지 꼭 확인!

개념 2
인수분해로 풀기
땡중샘

이차방정식 푸는 첫 번째 방법 — 인수분해야. 핵심 원리 먼저.

AB = 0 이면 A = 0 또는 B = 0
두 수의 곱이 0이 되려면 둘 중 하나는 반드시 0이어야 해.

태린

그걸로 이차방정식을 어떻게 풀어요?

땡중샘

x² − 5x + 6 = 0 풀어볼게. 두 수의 곱이 6이고 합이 −5인 수는 −2와 −3이야.

출발
x² − 5x + 6 = 0
인수분해
(x − 2)(x − 3) = 0
원리 적용
x − 2 = 0  또는  x − 3 = 0
해
x = 2  또는  x = 3
땡중샘

자주 나오는 세 가지 패턴 정리해줄게.

공통인수
x² − 4x = 0 → x(x−4) = 0 → x = 0 또는 4
합차공식
x² − 9 = 0 → (x+3)(x−3) = 0 → x = ±3
일반형
x² + x − 6 = 0 → (x+3)(x−2) = 0 → x = −3 또는 2
땡중샘

자주 하는 실수 하나!

x² − 5x = 0에서 양변을 x로 나누면 절대 안 돼!
x = 0인 해를 잃어버려. 반드시 x(x−5) = 0으로 인수분해해야 해.

땡중샘

확인 문제!

CHECK
x² − 5x + 4 = 0의 해는?
땡중샘

곱해서 4, 더해서 −5인 두 수 → −1과 −4. (x−1)(x−4) = 0 → x = 1 또는 x = 4!

개념 3
제곱근으로 풀기 (완전제곱식)
땡중샘

인수분해가 안 되는 경우엔 제곱근을 이용해. 기본 형태부터.

x² = k  →  x = ±√k   (k ≥ 0일 때)

(x − p)² = q  →  x = p ± √q

태린

(x+2)² = 9 같은 거도 돼요?

땡중샘

응! (x+2)² = 9이면 x+2 = ±3이야. 경우를 나눠서 풀어.

출발
(x+2)² = 9
제곱근
x+2 = ±3
경우 1
x = 3 − 2 = 1
경우 2
x = −3 − 2 = −5
땡중샘

그런데 x² + 6x + 1 = 0처럼 처음부터 완전제곱식이 아닌 경우는?
완전제곱식으로 만들어야 해.

x² + bx 에  (b/2)²을 더하면 완전제곱식!
x² + 6x + 9 = (x+3)²   ← 6÷2=3, 3²=9

땡중샘

x² + 6x + 1 = 0을 완전제곱식으로 풀어볼게.

① 상수 이항
x² + 6x = −1
② 9 더하기
x² + 6x + 9 = −1 + 9
③ 완전제곱
(x+3)² = 8
④ 제곱근
x+3 = ±2√2
⑤ 해
x = −3 ± 2√2
땡중샘

확인 문제!

CHECK
(x − 3)² = 7의 해는?
땡중샘

(x−3)² = 7 → x−3 = ±√7 → x = 3 ± √7. p = 3이니까 부호 주의!

개념 4
근의 공식
땡중샘

완전제곱식 방법을 ax² + bx + c = 0에 일반적으로 적용하면 공식이 나와. 이게 근의 공식이야.

x = −b ± √(b²−4ac) 2a

ax²+bx+c=0 (a≠0)에서 a, b, c 값을 공식에 대입하면 바로 해가 나와.

태린

공식이 복잡해 보여요…

땡중샘

차근차근 쓰면 어렵지 않아. 2x² + 3x − 2 = 0 같이 풀어볼게.

a, b, c 확인
a=2,  b=3,  c=−2
b²−4ac
3² − 4×2×(−2) = 9 + 16 = 25
공식 대입
x = (−3 ± √25) ÷ (2×2) = (−3 ± 5) ÷ 4
경우 1
x = (−3 + 5) ÷ 4 = 1/2
경우 2
x = (−3 − 5) ÷ 4 = −2
땡중샘

공식 안의 b² − 4ac를 판별식 D라고 해. D 값으로 근의 개수를 먼저 알 수 있어.

D > 0
서로 다른 두 실근
D = 0
중근 — 두 근이 같음
D < 0
실근 없음 (√ 안이 음수)
태린

중근이 뭐예요?

땡중샘

두 근이 같아지는 경우야. 예를 들어 x² − 6x + 9 = 0.
D = 36 − 36 = 0 → (x−3)² = 0 → x = 3 (중근).

땡중샘

확인 문제!

CHECK
x² − 4x + 1 = 0을 근의 공식으로 풀면?
땡중샘

a=1, b=−4, c=1. D = 16−4 = 12.
x = (4 ± √12) / 2 = (4 ± 2√3) / 2 = 2 ± √3!

이차방정식 활용 문제는 무조건 4단계로 풀어.

① 설정

② 세우기
조건을 이차방정식으로 표현
③ 풀기
이차방정식의 근 구하기
④ 검토
근이 문제 조건에 맞는지 확인!
땡중샘

예제 1 — 연속하는 두 자연수
연속하는 두 자연수의 곱이 42야. 두 수를 구해봐.

태린

작은 수를 x라 하면 큰 수는 x+1이요!

땡중샘

딱 맞아! x(x+1) = 42로 방정식을 세워서 풀면:

방정식
x(x+1) = 42
전개
x² + x − 42 = 0
인수분해
(x+7)(x−6) = 0
근
x = −7 또는 x = 6
✔ 검토
자연수 조건 → x = 6, 두 수는 6과 7
땡중샘

예제 2 — 넓이 문제
가로가 세로보다 3cm 더 긴 직사각형의 넓이가 40cm²야. 세로의 길이를 구해봐.

태린

세로를 x, 가로를 x+3으로 놓으면 되죠!

땡중샘

완벽해. x(x+3) = 40이야.

방정식
x(x+3) = 40
전개
x² + 3x − 40 = 0
인수분해
(x+8)(x−5) = 0
근
x = −8 또는 x = 5
✔ 검토
길이는 양수 → 세로 = 5cm, 가로 = 8cm
땡중샘

확인 문제!

CHECK
연속하는 두 자연수의 곱이 30일 때, 두 수의 합은?
땡중샘

x(x+1) = 30 → x²+x−30 = 0 → (x+6)(x−5) = 0 → x = 5.
두 수는 5와 6 → 합 = 11!
