개념 1
줄기와 잎 그림
땡중샘

태린아, 수학 점수 10개를 받았어. 그냥 나열하면 뭐가 보여?

72, 85, 91, 68, 75, 83, 90, 64, 77, 88
태린

숫자들이 많아서 잘 모르겠는데요. 뭔가 규칙이 있는 것 같기도 하고...

땡중샘

맞아. 그냥 나열하면 패턴을 알기 어려워. 그래서 줄기와 잎 그림을 써. 십의 자리를 줄기, 일의 자리를 잎으로 분리하는 거야.

나무를 생각해봐. 줄기 하나에 잎이 여러 개 달리듯이 — 줄기 6에는 4, 8이 달리고, 줄기 7에는 2, 5, 7이 달리는 식이야.

태린

아, 줄기가 뭐고 잎이 뭐예요? 잘 모르겠어요.

땡중샘

예를 들어 72라는 수에서 십의 자리 7이 줄기, 일의 자리 2가 잎이야. 64는 줄기 6, 잎 4가 되는 거지. 10개 자료를 정리하면 이렇게 돼.

줄기잎
64   8
72   5   7
83   5   8
90   1

줄기와 잎 그림의 장점
원래 자료값이 그대로 보존돼서 데이터를 잃지 않아. 또 분포 모양을 한눈에 파악할 수 있어 — 어느 구간에 자료가 몰려있는지 바로 보여.

태린

잎을 보면 작은 수부터 순서대로 적혀있네요. 그러면 중앙값도 쉽게 찾을 수 있겠어요!

땡중샘

바로 그거야. 잎을 순서대로 적어두면 중앙값, 최솟값, 최댓값을 바로 찾을 수 있어. 자, 확인 문제!

확인 문제
위 줄기와 잎 그림에서 중앙값(5번째와 6번째 값의 평균)은?
자료: 64, 68, 72, 75, 77, 83, 85, 88, 90, 91 (오름차순)
수정 요청
개념 2
도수분포표
땡중샘

자료가 많아지면 줄기와 잎 그림도 복잡해져. 그래서 쓰는 게 도수분포표야. 먼저 용어 정리부터.

변량자료에서 조사한 값 (예: 수학 점수 72, 85, ...)
계급변량을 나눈 구간 (예: 60이상 70미만)
계급의 크기계급의 양 끝값 차 (예: 70 - 60 = 10)
계급값계급의 양 끝값의 평균 (예: (60+70)÷2 = 65)
도수각 계급에 속하는 자료의 수 (개수)
상대도수도수 ÷ 전체 도수 (비율 — 0~1 사이)
태린

계급 크기를 왜 일정하게 해요? 마음대로 정하면 안 돼요?

땡중샘

크기가 다르면 막대 너비가 달라져서 비교가 어려워. 예를 들어 한 계급이 10짜리고 다른 계급이 20짜리면, 20짜리가 더 넓어서 도수가 커 보일 수 있어. 일정하게 해야 공평하게 비교할 수 있는 거야.

땡중샘

아까 10개 점수로 도수분포표를 만들면 이렇게 돼. 전체 도수는 10이야.

계급 (점) 계급값 도수 (명) 상대도수
60이상 70미만6520.2
70이상 80미만7530.3
80이상 90미만8530.3
90이상 100미만9520.2
합계101.0
태린

상대도수가 도수를 전체로 나눈 거면, 합하면 항상 1이 되는 거네요!

땡중샘

정확해. 상대도수의 합은 항상 1이야. 비율이니까. 자, 확인 문제!

확인 문제
전체 도수가 25명이고, 어떤 계급의 도수가 5명이라면 상대도수는?
수정 요청
개념 3
히스토그램과 도수분포다각형
땡중샘

도수분포표를 그래프로 나타낸 게 히스토그램이야. 막대가 붙어 있어서 일반 막대그래프와 달라 보여.

태린

히스토그램이랑 막대그래프랑 다른 게 뭐예요? 막대가 붙어있는 것 말고요.

땡중샘

일반 막대그래프는 각 항목이 독립적이야 — 예를 들어 국어, 수학, 영어 점수 비교. 근데 히스토그램은 연속적인 수치 자료를 나타내. 점수가 60에서 100까지 이어지는 것처럼. 막대를 붙이는 건 계급이 연속된다는 뜻이야.

막대그래프항목 간 간격 있음, 독립적 범주 비교
히스토그램막대 간 간격 없음, 연속 수치 자료 표현
땡중샘

도수분포다각형은 히스토그램 각 막대의 꼭대기 중점을 선으로 연결한 그래프야. 양 끝에는 도수가 0인 계급을 하나씩 추가해서 선이 x축과 만나게 해줘.

도수분포다각형 그리는 법
1. 각 계급 막대의 꼭대기 중점에 점을 찍는다.
2. 양 끝에 도수 0인 가상 계급을 추가한다.
3. 모든 점을 직선으로 연결한다.

태린

도수분포다각형은 히스토그램이랑 같은 정보를 담고 있는 거 아닌가요?

땡중샘

맞아. 같은 정보를 다른 방식으로 표현한 거야. 근데 다각형이 편리한 점이 있어 — 두 집단을 같은 그래프에 겹쳐 그릴 수 있어. 히스토그램은 막대가 겹쳐서 보기 어렵거든. 자, 확인 문제!

확인 문제
위 히스토그램에서 80점 이상 90점 미만인 학생 수는?
(도수분포표: 60~70: 2명, 70~80: 3명, 80~90: 3명, 90~100: 2명)
수정 요청
개념 4
상대도수의 분포
땡중샘

상대도수에 대해 좀 더 깊이 들어가 보자. 상대도수의 합은?

태린

항상 1이라고 하셨잖아요!

땡중샘

맞아. 항상 1이야. 그럼 왜 도수 대신 상대도수를 쓰는 걸까? 예를 들어 1반(30명)과 2반(40명)의 수학 성적을 비교한다고 해봐.

태린

왜 도수 대신 상대도수를 써요? 집단 크기가 다를 때 뭔가 문제가 있어요?

땡중샘

좋은 질문이야. 1반에서 80점 이상이 12명, 2반에서 80점 이상이 14명이면 2반이 더 잘한 것 같지? 근데 1반은 30명, 2반은 40명이야. 상대도수로 보면 달라.

1반 (30명)2반 (40명)
80점 이상 도수12명14명
상대도수12÷30 = 0.414÷40 = 0.35

상대도수로 보면 1반(0.4)이 2반(0.35)보다 80점 이상 비율이 더 높아. 전체 수가 다를 때는 도수만 보면 오해할 수 있어.

땡중샘

누적상대도수는 처음 계급부터 해당 계급까지의 상대도수를 모두 더한 값이야. 예를 들어 70점 미만인 학생의 비율을 알고 싶을 때 써.

60~70상대도수 0.2  →  누적 0.2
70~80상대도수 0.3  →  누적 0.2+0.3 = 0.5
80~90상대도수 0.3  →  누적 0.5+0.3 = 0.8
90~100상대도수 0.2  →  누적 0.8+0.2 = 1.0
땡중샘

자료의 해석 — 대표값
땡중샘

자료를 하나의 값으로 대표할 때 쓰는 게 대표값이야. 세 가지를 알아야 해.

평균전체 합 ÷ 개수    — 모든 값을 반영
중앙값크기순 나열 시 가운데 값    — 극단값에 덜 영향받음
최빈값가장 많이 나타나는 값    — 여러 개일 수도 있음
태린

평균이 대표값인데 중앙값을 왜 따로 써요?

땡중샘

예를 들어볼게. 5명이 사는 동네에서 연봉을 조사했어.

3000만, 3500만, 4000만, 4500만, 100000만

평균: (3000+3500+4000+4500+100000) ÷ 5 = 23000만원

중앙값: 크기순 가운데 → 4000만원

태린

억만장자 한 명 때문에 평균이 23000만원이 됐어요? 그럼 대부분의 사람은 평균보다 훨씬 적게 버는 건데...

땡중샘

바로 그거야. 평균은 극단값(outlier)에 크게 영향받아. 이런 경우엔 중앙값(4000만원)이 실제 분포를 더 잘 나타내. 신문에서 "평균 연봉 5000만원"이라고 해도 중앙값은 훨씬 낮을 수 있어.

평균이 적합극단값 없는 고른 분포 자료
중앙값이 적합극단값이 있거나 분포가 치우친 자료
최빈값이 적합수치보다 범주형 자료 (예: 선호 색상)
땡중샘
