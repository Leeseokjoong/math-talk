개념 1
제곱수와 제곱근의 뜻
땡중샘

태린아, 3을 두 번 곱하면 뭐야?

태린

9요! 3 × 3 = 9.

땡중샘

맞아. 이렇게 같은 수를 두 번 곱해서 나오는 수를 제곱수라고 해.

1² = 1   2² = 4   3² = 9   4² = 16   5² = 25
6² = 36   7² = 49   8² = 64   9² = 81   10² = 100

이 숫자들 꼭 외워둬. 문제마다 나와.

땡중샘

이제 반대 방향. 9는 뭘 두 번 곱해서 만든 거야? 이 답을 구하는 게 제곱근이야.

x² = a 를 만족하는 x를 a의 제곱근이라 한다.
9의 제곱근 → 3² = 9이니까 x = 3. 근데 (−3)² = 9도 맞아!

태린

그럼 9의 제곱근이 두 개예요?

땡중샘

응! 양수의 제곱근은 항상 두 개야. 경우를 정리하면:

양수 a
제곱근이 2개 — 양수 하나 + 음수 하나 → ±√a
0
제곱근이 1개 — 0만 (0² = 0)
음수
제곱근이 없음 — 두 번 곱해서 음수는 불가능
땡중샘

확인 문제!

CHECK
36의 양의 제곱근은?
땡중샘

36의 제곱근은 ±6 두 개인데, 양의 제곱근은 +6 하나야. "양의"라는 말에 주목!

제곱근
x² = a를 만족하는 x
양수의 제곱근: ±√a (두 개)
제곱수: 1,4,9,16,25,36,49,64,81,100 — 외워두자!
수정 요청 메모
개념 2
제곱근의 성질
땡중샘

√(루트) 기호 알지? √a는 a의 양의 제곱근이야. 양수 하나만 나타내.

√9 = 3    (양의 제곱근만)
±√9 = ±3   (양수·음수 둘 다)
땡중샘

중요한 성질 두 가지야.

성질 1
(√a)² = a   (a ≥ 0)
제곱근을 두 번 제곱하면 원래 수
성질 2
√(a²) = a   (a ≥ 0)
제곱한 수의 루트는 원래 수

예: (√5)² = 5, √(3²) = √9 = 3

태린

√(−3)²는 뭐예요? −3이에요?

땡중샘

좋은 질문! 이게 헷갈리는 부분이야.

√(−3)² = √9 = 3  (−3이 아니야!)

√(a²) = |a| — 절댓값이야. 결과는 항상 0 이상이야.
a가 음수여도 √(a²)는 −a (양수)가 돼.

a ≥ 0일 때
√(a²) = a
a < 0일 때
√(a²) = −a  (양수)
땡중샘

확인 문제!

CHECK
√(−5)²의 값은?
땡중샘

√(−5)² = √25 = 5. 루트 안을 먼저 계산해! (−5)² = 25이고, √25 = 5야.

수정 요청 메모
개념 3
무리수와 실수
땡중샘

√2를 계산기로 누르면 1.41421356… 소수가 끝나지 않고 반복도 안 해. 이런 수를 무리수라고 해.

유리수: 분수 ab (b≠0, a,b 정수)로 나타낼 수 있는 수
→ 정수, 유한소수, 순환소수 포함

무리수: 분수로 나타낼 수 없는 수
→ √2, √3, √5, π 등 — 소수가 끝없이, 반복 없이 계속됨

태린

√4는요? 2잖아요.

땡중샘

맞아! √4 = 2야. 딱 떨어지니까 유리수야. 제곱수의 루트는 유리수, 나머지는 무리수.

유리수
√1=1, √4=2, √9=3, √16=4, √25=5 …
무리수
√2, √3, √5, √6, √7, √8, √10 …
땡중샘

유리수 + 무리수 = 실수. 우리가 수직선 위에 나타낼 수 있는 모든 수야.

실수
├유리수정수, 유한소수, 순환소수
└무리수√2, √3, π …

수정 요청 메모
개념 4
근호 포함 식의 계산
땡중샘

루트가 들어간 식을 계산하는 규칙이 있어.

√a × √b = √(ab)   (a,b ≥ 0)
√a ÷ √b = √(a/b)   (a ≥ 0, b > 0)
k√a × m√b = km√(ab)

땡중샘

그리고 루트 안에 제곱수가 있으면 밖으로 꺼낼 수 있어.

예 1
√12 = √(4×3) = √4 × √3 = 2√3
예 2
√18 = √(9×2) = √9 × √2 = 3√2
예 3
√2 × √3 = √6
예 4
2√3 × 3√2 = 2×3 × √(3×2) = 6√6
태린

루트 안에서 제일 큰 제곱수를 찾아서 꺼내면 되는 거죠?

땡중샘

완벽해! 가장 큰 제곱수로 인수분해하는 게 핵심이야.
√48 = √(16×3) = 4√3  ←  4가 아니라 16으로 꺼내야 해. 4로 꺼내면 √48=2√12로 아직 정리가 안 된 거야.

√48 = 2√12 → 아직 12 안에 4가 있어서 더 정리해야 해.
처음부터 최대 제곱수(16)를 찾아서 4√3으로 한 번에 끝내자.

땡중샘

확인 문제!

CHECK
√12 × √3의 값은?
땡중샘

√12 × √3 = √36 = 6! 또는 √12 = 2√3이니까 2√3 × √3 = 2×3 = 6이야.

개념 5
분모의 유리화
땡중샘

1√2 같은 수, 분모에 루트가 있으면 계산이 불편해. 이걸 분모에서 없애는 걸 유리화라고 해.

분자·분모에 분모와 같은 √를 곱하면 분모가 유리수가 돼.
√a × √a = a 성질을 이용하는 거야.

땡중샘

예제들 같이 풀어볼게.

예 1
1√2 = 1 × √2√2 × √2 = √22 = √2/2
예 2
3√5 = 3√55 = 3√5/5
예 3
6√3 = 6√33 = 2√3
태린

분모에 2√3처럼 계수가 있으면요?

땡중샘

루트 부분만 유리화하면 돼.

예 4
52√3 = 5 × √32√3 × √3 = 5√36 = 5√3/6

2×3=6이니까 분모가 6이 되는 거야.

땡중샘

땡중샘

4/√2 = 4√2/2 = 2√2. 4를 2로 약분하면 돼! ④도 틀리지 않지만 약분 전이라 최종 답은 ③이야.
