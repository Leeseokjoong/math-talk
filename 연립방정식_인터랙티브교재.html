태린아, 오늘부터 연립방정식이야. 워밍업 문제 하나 — x + y = 5에서 x = 2, y = 3이면 해야?

태린

네, 2 + 3 = 5니까 맞잖아요!

땡중샘

맞아. 근데 x = 1, y = 4도 맞아. x = 0, y = 5도 맞아. x = 100, y = −95도 맞아.

태린

어? 그럼 답이 엄청 많은 거예요?

땡중샘

맞아. 조건이 하나뿐이면 해가 무한히 많아. x랑 y를 딱 하나로 정하려면 조건이 하나 더 필요해.

ax + by = c 형태 — 이게 미지수가 2개인 일차방정식이야. 해가 무한히 많아서 두 번째 방정식이 반드시 있어야 해.

연립방정식의 뜻과 해
땡중샘

방정식이 두 개면 뭐가 달라질까? 예를 들어 이 두 개를 동시에 만족해야 한다면?

x + y = 5   … ①
x − y = 1   … ②
태린

둘 다 동시에 만족한다는 게 무슨 말이에요?

땡중샘

같은 x, y를 넣었을 때 ①도 성립하고 ②도 성립하는 거야. x = 3, y = 2를 넣어봐.

태린

3 + 2 = 5 맞고, 3 − 2 = 1 맞아요! 둘 다 성립하네요!

땡중샘

그게 바로 연립방정식의 해야. 두 방정식을 동시에 만족하는 x, y의 값을 해라고 해.

방정식 두 개를 한 쌍으로 묶은 것 → 연립방정식. 두 조건을 동시에 만족해야 하니까 해가 딱 하나로 좁혀지는 거야.

연립방정식의 해

두 방정식을 동시에 만족하는 x, y의 값

가감법 — 더하거나 빼서 하나를 없애라
땡중샘

이제 연립방정식 풀기 시작이야. 첫 번째 방법 — 가감법. 두 방정식을 더하거나 빼서 미지수 하나를 없애는 거야.

태린

더하거나 빼면 미지수가 없어져요? 어떻게요?

땡중샘

이 두 식을 더해봐. y 계수가 +1, −1이야.

x + y = 5  … ①
x − y = 1  … ②
땡중샘

①+②를 하면 y가 사라져!

①+②2x = 6  →  x = 3
①에 대입3 + y = 5  →  y = 2
해x = 3,  y = 2
태린

오! y가 사라지니까 x만 남아서 풀 수 있네요!

땡중샘

그런데 계수가 같지 않으면 어떡해? 그럴 땐 배수로 맞춰야 해.

2x + 3y = 11  … ①
x + 2y = 7   … ②
땡중샘

②에 2를 곱하면 x 계수가 같아져서 빼면 x가 사라져.

②×22x + 4y = 14  … ②'
①−②'−y = −3  →  y = 3
②에 대입x + 2(3) = 7  →  x = 1
해x = 1,  y = 3
태린

배수로 맞추는 거 생각보다 어렵지 않네요. 계수를 같게 만들면 되는 거잖아요!

땡중샘

정확해! 확인 문제 풀어봐.

개념 4
대입법 — 한 식을 다른 식에 꽂아 넣는다
땡중샘

두 번째 방법 — 대입법이야. 이름 그대로 대입하는 거야.

태린

대입이요? 꽂아 넣는다는 말이에요?

땡중샘

맞아. 한 방정식에서 y = 뭔가 형태로 만들고, 다른 방정식의 y 자리에 그걸 꽂아 넣는 거야.

y = 2x − 1  … ①
3x + y = 9  … ②
땡중샘

①이 이미 y = 형태야. ②의 y 자리에 (2x−1)을 넣어봐.

①을 ②에3x + (2x−1) = 9
5x − 1 = 9  →  x = 2
①에 대입y = 2(2) − 1 = 3
해x = 2,  y = 3
태린

y 자리에 넣으면 x만 남으니까 풀 수 있네요! 근데 y = 형태가 아니면요?

땡중샘

그럼 먼저 변형하면 돼. x + y = 6 이면 → y = 6 − x 로 만들고 대입.

그럼 가감법 vs 대입법, 언제 어떤 걸 써?

가감법이 편할 때
계수가 같거나 배수 관계일 때
예: 2x+y=7 / x−y=2
대입법이 편할 때
이미 y= 또는 x= 형태일 때
예: y=2x−1 / 3x+y=9
땡중샘

선생님, 시험 문제 보면 소수 계수도 나오고 분수도 나오던데... 저것도 다 연립방정식이에요?

땡중샘

맞아. 시험엔 일부러 복잡하게 만들어서 내. 근데 패턴은 딱 세 가지야. 각각 먼저 정리하는 방법만 알면 나머지는 똑같아.

땡중샘

① 괄호가 있을 때 — 분배법칙으로 먼저 펼치기

2(x+1) + y = 9
→ 2x + 2 + y = 9
→ 2x + y = 7
땡중샘

② 소수 계수가 있을 때 — 10 또는 100을 곱해서 정수로

0.2x + 0.3y = 1.4
→ 양변 ×10: 2x + 3y = 14
땡중샘

③ 분수 계수가 있을 때 — 분모의 최소공배수를 곱해서 정수로

12x + 13y = 2
→ 양변 ×6 (분모 2, 3의 최소공배수): 3x + 2y = 12
태린

결국 먼저 깔끔하게 만들고, 그 다음에 가감법이나 대입법으로 푸는 거군요!

땡중샘
